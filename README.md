# 💰 AI Finance Bot — Telegram + n8n + GPT-4o

Персональный финансовый помощник в Telegram. Записывает доходы и расходы голосом или текстом, считает баланс, строит отчёты и хранит всё в Google Sheets.

---

## Что умеет

- **Запись голосом** — отправляешь голосовое, бот распознаёт через Whisper и записывает
- **Запись текстом** — пишешь в свободной форме: `потратил 500 на еду`, `получил зарплату 80000`
- **Баланс** — показывает текущий баланс в любой момент
- **Отчёт** — разбивка расходов по категориям за период
- **История** — список последних операций
- **Цели** — можно поставить финансовую цель и отслеживать прогресс
- **Удаление записей** — с подтверждением через inline-кнопки
- **Память** — бот помнит контекст разговора (Postgres)

---

## Стек

| Инструмент | Для чего |

| n8n | Оркестрация всего воркфлоу |
| GPT-4o | Понимание текста, классификация операций |

| PostgreSQL | Память контекста диалога |

## Архитектура

```
Telegram (текст/голос/кнопки)
    ↓
n8n Webhook Trigger
    ↓
Маршрутизатор (Switch) — 9 веток
    ↓
┌─────────────────────────────────┐
│  /start → приветствие           │
│  Текст → ИИ Агент (GPT-4o)      │
│  Голос → Whisper → ИИ Агент     │
│  Кнопки → ИИ Агент              │
│  Callback → подтверждение       │
└─────────────────────────────────┘
    ↓
ИИ Агент с инструментами:
  • create_new_register
  • get_all_registers
  • update_register
  • delete_register
  • Calculator
    ↓
Google Sheets (база данных)
    ↓
Ответ в Telegram
```

## Структура Google Sheets

| Колонка | Описание |
|---|---|
| id | Уникальный ID записи (timestamp) |
| user_id | Telegram ID пользователя |
| type | `income` или `expense` |
| value | Сумма |
| category | Food, Transport, Salary и др. |
| payment_method | cash, card и др. |
| description | Краткое описание |
| date | Дата операции |

## Установка

> ⚠️ Этот воркфлоу создан под конкретную инфраструктуру (self-hosted n8n + Postgres + Google Sheets). Для запуска нужно настроить свои credentials.

1. Импортируй `finance-bot-n8n.json` в свой n8n
2. Создай credentials:
   - Telegram Bot API
   - OpenAI API
   - Google Sheets OAuth2
   - PostgreSQL
3. Замени `YOUR_GOOGLE_SHEET_ID` на ID своей таблицы
4. Создай Google Sheets с колонками из таблицы выше
5. Активируй воркфлоу
